# 简介
本项目是北京科技大学小学期课程设计的内容，目的是使用jsp jdbc等技术实现一个简单的web版银行管理系统。

本项目适合**Jsp新手作为入门项目，逻辑比较简单，不涉及前端界面复杂的操作**

内容包括：**数据展示、jdbc增删改查的操作**

# 项目运行

 1. clone项目到本地 
 2. 在mysql中执行sql包下的bank.sql文件
 3.  配置Tomcat运行项目
